# Anubis

Anubis standalone cheat client based on Meteor Client.

## Metadata

- Name: Anubis
- Developer: Pedro
- Mod ID: `anubis`
- Package: `com.pedro.anubis`
- Minecraft: `1.21.4`
- Meteor Client: `1.21.4-SNAPSHOT`

## Development

- Run the `Minecraft Client` configuration from your IDE.
- Build with `./gradlew build`.
- Output jar is generated under `build/libs`.

## Notes

- Replace `src/main/resources/assets/anubis/icon.png` with your own icon if needed.
- Repository metadata points to `https://github.com/Pedro/Anubis`.
